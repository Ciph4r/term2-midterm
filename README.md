## Routes Base Path: /api/v1

### User routes /user
1. post `/register`
2. get `/register-update/:name/:password`
3. get `/login`
4. post `/login`
5. get `/logout`
6. get `/account`
7. put `/account`
8. get `/profile`
9. put `/profile`
10. get `/profileEdit`
11. get `/add-goal`
12. put `/add-goal`
13. put `/weigh-in`

### Diet Plan routes /dietplan

1. get `/`
2. post `/add-diet`
3. get `/add-meal`
4. get `/show-meals/:dietPlan_id`
5. get `/food-search/:meals_id`
6. get `/findfood`
7. put `/add-food`
8. put `/remove-item/:meal_id`
